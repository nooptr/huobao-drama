# 风格分析指南

## 分析维度

1. **叙事风格**：现实主义/浪漫/悬疑/喜剧/奇幻
2. **视觉调性**：明亮温暖/阴暗冷峻/柔和梦幻/浓烈饱和
3. **色彩倾向**：暖色调/冷色调/单色系/高对比
4. **节奏感**：快节奏剪辑/慢镜头抒情/张弛有度

## 可用风格

| 风格值 | 描述 |
|--------|------|
| realistic | 真实感，电影质感，自然光影 |
| ghibli | 吉卜力风格，温暖手绘质感 |
| anime | 日系动漫风格，线条清晰 |
| watercolor | 水彩风格，柔和朦胧 |
| comic | 漫画风格，强对比线条 |
| cinematic | 电影大片质感，戏剧性光影 |
| noir | 黑色电影，高对比黑白 |
| fantasy | 奇幻风格，魔幻色彩 |

## 分析步骤

1. 调用 `read_episode_content` 获取剧本内容和当前风格设置
2. 分析文本中的环境描写、情绪基调、时代背景
3. 根据分析结果推荐最合适的风格
4. 如需更新，调用 `update_drama_style` 应用新风格
